SetDefault(K, V) Method
DictionariesSetDefaultK, V Method
Returns a value from dictionary by the key. If key is not defined yet then a new default entry is created and returned.

Usage Note

Examples
It's most useful when dealing with dictionaries of a complex type:
Copy
1var a = new Dictionary<int, HashSet<string>>();
2// An empty string set for key 1 is created, and "abc" is added in it.
3a.SetDefault(1).Add("abc");
4// "def" is added into existing string set at key 1. 
5a.SetDefault(1).Add("def");
