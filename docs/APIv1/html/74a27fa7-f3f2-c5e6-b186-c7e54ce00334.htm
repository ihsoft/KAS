Creates a primitive mesh and attaches it to the model.

public static GameObject CreatePrimitive(
	PrimitiveType type,

)

Parameters

type
Type: PrimitiveType
Type of the primitive.
meshScale
Type: Vector3
Scale to bring all mesh vertices to. New primitive have base size of 1m but some shapes may have exceptions (e.g. height of a cylinder is 2m). The scale is applied on the mesh, i.e. it's applied on the vertices, not the transform.
material
Type: Material
Material to use for the primitive.
parent
Type: Transform
Parent transform to attach primitive to.

Return Value

Type: GameObject
Game object of the new primitive.
Remarks
For Cylinder Z and Y axis will be swapped to make Z "the length".

Collider on the primitive will be destroyed. Consider using AdjustCollider(GameObject, PrimitiveType, Vector3, CollidersPrimitiveCollider) to setup the right collider when needed.
