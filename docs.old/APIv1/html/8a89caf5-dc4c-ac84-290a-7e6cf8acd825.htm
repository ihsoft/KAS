Methods
  NameDescription
CallOnEndOfFrame
Delays execution of the delegate till the end of the current frame.
CallOnFixedUpdate
Delays execution of the delegate till the next fixed update.
CallOnTimeout
Delays execution of the delegate for the specified amount of time.
Top
Remarks
Use theese tools when code needs to be executed right with some delay or at a specific moment of the game.
