Remarks
Use this player when soucre of the sound is player keyboard actions or a mouse pointer. This class implements all the boilerplate to load and play sound resources.
Examples
Here is an example of playing two different sounds on pressing "O" or "P" keys.
Copy
 1class MyModule : PartModule {
 2  public override OnAwake() {
 3    // We don't want to loose latency on "ooo.ogg" playing.
 4    UISoundPlayer.instance.CacheSound("ooo.ogg");
 5  }
 6
 7  public override OnUpdate() {
 8    if (Input.GetKeyDown("O")) {
 9      UISoundPlayer.instance.Play("ooo.ogg");  // Played from cache. No delay.
10    }
11    if (Input.GetKeyDown("P")) {
12      UISoundPlayer.instance.Play("ppp.ogg");  // May delay game while loading the resource.
13    }
14  }
15}
